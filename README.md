# povvo community health

This repository supplies the default community health files for public repositories owned by [@povvo](https://github.com/povvo).

GitHub inherits these contribution, governance, funding, security, support, issue, and pull-request defaults only when a repository does not provide its own file of the same type. Repository-specific licences, privacy policies, terms, documentation, and overrides remain with each project.

The `bug` and `enhancement` labels used by the shared issue forms must also exist in each repository that inherits those forms.
